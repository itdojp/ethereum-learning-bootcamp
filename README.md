# ethereum-learning-bootcamp

14日間でEthereum開発の基礎から応用まで学ぶ完全教材。  
本リポジトリは教育目的・非商用利用を前提としています。

## 構成
- `curriculum/` : Day01〜Day14 各日の教科書＋ハンズオン
- `appendix/` : 補足資料（EIP, リソース集）
- `contracts/`, `scripts/`, `test/`: 実践サンプル
- `.github/workflows/`: CI/CD雛形
